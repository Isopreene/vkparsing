Парсер групп вКонтакте

Привет! Это парсер для групп vk.com

Пошлите get-запрос вида {{ host }}/?params. Список доступных параметров:
  • groups=(через запятую)
  • all=(True/False)
  • date_start=(YYYY-MM-DD)
  • date_finish=(YYYY-MM-DD)

    • Запрос {{host}}/all работает как {{host}}/?all=True

    {{ host }}/?groups=animatron
    Проверить группу vk.com/animatron.
    Взять последние 100 записей группы
    {{ host }}/?groups=teplomult,animatron,public218140062
    Проверить группы vk.com/teplomult, vk.com/animatron, vk.com/public218140062.
    Взять последние 100 записей группы
    {{ host }}/?groups=animatron&date_start=2023-01-012
    Проверить группу vk.com/animatron.
    Взять последние 100 записей с 1 января 2023 года по текущий день
    {{ host }}/?groups=animatron&date_start=2023-01-01&date_finish=2023-04-04
    Проверить группу vk.com/animatron.
    Взять последние 100 записей с 1 января 2023 года по 4 апреля 2023 года
    {{ host }}/all
    {{ host }}/?all=True
    Взять последние 100 записей каждой группы, хранящейся в базе данных
